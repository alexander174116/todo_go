CLI-приложение на Go.
Описание функций:
1) Добавить задачу → add --desc="описание задачи". Далее задаче присваивается уникальный ID.
2) Вывести отфильтрованный список задач → list --filter=all/done/pending/ Функция all - отображает список всех задач, функция done отобразит список выполненных задач, а pending - отобразит список невыполненных задач.
3) Отметить задачу как выполненную → complete --id=[id задачи, которую нужно отметить выполненной]
4) Полностью удалить задачу из списка всех задач → delete --id=[id задачи, которую нужно удалить]
5) Экспортировать текущий список всех задач в файл. Доступные форматы: CSV/JSON. Использование: export --format=[csv/json] --out=[имя файла, например: backup.json]
6) Импортировать задачи из файла (заменяет весь текущий список задач), доступные форматы: SCV/JSONю Использование: load --file=[имя/путь файла, откуда импортировать задачи]
